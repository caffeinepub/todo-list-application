# Todo List Application

## Overview
A simple todo list application that allows users to manage their tasks with full CRUD operations and filtering capabilities.

## Core Features

### Task Management
- Add new tasks with title and optional due date
- Edit existing task titles and due dates
- Delete tasks
- Mark tasks as complete or incomplete
- Tasks persist between browser sessions

### Task Display
- Display all tasks in a clean list view
- Show task title, completion status, and due date (if set)
- Visual distinction between completed and active tasks

### Filtering
- Filter tasks by status:
  - All tasks
  - Active (incomplete) tasks only
  - Completed tasks only

### User Interface
- Simple and clean design
- Intuitive task management controls
- Clear visual feedback for task states
- Responsive layout

## Data Storage
The backend stores:
- Task data including title, completion status, and due date
- Persistent storage ensures data survives browser refresh

## Backend Operations
- Create new tasks
- Retrieve all tasks
- Update task details (title, completion status, due date)
- Delete tasks
